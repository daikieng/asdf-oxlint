# asdf-oxlint

[Oxlint](https://oxc.rs/docs/guide/usage/linter) plugin for [asdf](https://asdf-vm.com) version manager.

## Installation

```bash
asdf plugin-add oxlint https://github.com/daikieng/asdf-oxlint.git
```

## Usage

```bash
# Show all installable versions
asdf list-all oxlint

# Install latest version
asdf install oxlint latest

# Set a version globally
asdf global oxlint latest
```

Check [asdf](https://github.com/asdf-vm/asdf) readme for instructions on how to install & manage versions.
